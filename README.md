Edupay
Edupay is an innovative platform designed to support African children’s education through stable coins like USDT. Built on the robust technologies of Lisk and Base, Edupay leverages blockchain to create a secure and transparent way for well-wishers, parents, NGOs, and guardians to contribute to educational initiatives.

Introduction
Edupay aims to promote education in Africa by providing a platform where contributions can be made securely and transparently using blockchain technology. Our mission is to ensure that every donation directly benefits the children who need it most.

Features
Secure Donations: Utilize stable coins like USDT for contributions.
Transparency: Built on blockchain to ensure all transactions are transparent and traceable.
User-Friendly Interface: Developed with React and styled using Tailwind CSS for a seamless user experience.
Technologies Used
Blockchain: Lisk and Base
Frontend: React
Styling: Tailwind CSS
Installation
To get started with Edupay, follow these steps:

Clone the repository:
git clone https://github.com/yourusername/edupay.git
cd edupay

Install dependencies:
npm install

Start the development server:
npm start

Usage
Once the development server is running, you can access the platform at http://localhost:3000. From here, you can explore the features and functionalities of Edupay.
